# Flex Sensor Controlled Robotic Arm

Arduino-based robotic arm controlled using flex sensors and servo motors.  
The bending of the flex sensor changes resistance, which is read by Arduino and mapped to servo motor angles to mimic human hand motion.

## Hardware Components
- Arduino Uno
- Flex Sensor
- Servo Motors
- Breadboard
- Resistors
- Jumper wires
